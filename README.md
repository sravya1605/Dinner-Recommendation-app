# Dinner Recommendation App

## About
A mobile application built with React Native and Expo that helps couples decide what to have for dinner by recommending meals based on preferences, mood, and location. Designed with a clean, romantic UI using Playfair Display and Inter fonts.

---

## Features
- Personalized dinner recommendations for couples
- Location-aware suggestions using device GPS
- Smooth animations and haptic feedback for an engaging UX
- Bottom tab navigation with intuitive routing via Expo Router
- Gradient UI with blur effects for a polished look
- Cross-platform support — runs on iOS, Android, and Web

---

## Tech Stack

| Technology | Purpose |
|---|---|
| React Native 0.79 | Core mobile framework |
| Expo SDK 53 | Build and dev tooling |
| TypeScript | Type-safe development |
| Expo Router | File-based navigation |
| Expo Location | GPS-based suggestions |
| React Native Reanimated | Smooth animations |
| Lucide React Native | Icon library |
| Expo Linear Gradient | UI styling |
| Expo Camera | Camera integration |

## Getting Started

### Prerequisites
- Node.js >= 18
- Expo CLI installed globally

### Installation
```bash
git clone https://github.com/sravya1605/Dinner-Recommendation-app.git
cd Dinner-Recommendation-app
npm install
```

### Run the App
```bash
npm run dev
```
Scan the QR code with Expo Go on your phone, or press w to open in browser.

### Build for Web
```bash
npm run build:web
```
